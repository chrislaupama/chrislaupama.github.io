Scope

What 'scope' is, and how it works in JavaScript.

Scope is all about what code can be read dependant on where the code is defined. There's two types of scope, global and local.

Global scope is where code is defined outside of a function, thus allowing it to be called anywhere within the code.

Local scope is where things like variables are only accessible within the function (and any nested functions, which is functions within functions) it is decalred within, meaning it is now only accesible locally within that function and if you try and call that variable outside of the function it will come up undefined.

As an example, you may have a variable defined bigDogs = 10, but within a function you may have a variable that you also want to call bigDogs, this does not mean that bigDogs is equal to 10 as the functions own variables have their own scope. The same works vice versa, so if you have a variable in a function which says bigDogs = 15, that does not take over the value of the variable which is outside of the function it was defined in.
