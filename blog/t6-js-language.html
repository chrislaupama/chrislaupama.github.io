Javascript Language

Parentheses

Parentheses are used to hold information or data that is used to then utilise that information in conjunction with other operations.

Brackets

Brackets are used for arrays within JS. Arrarys are like a folder/bucket of information, so that they can be called or used later on.

Braces

Braces are used to hold information within an operation, like, if there is a function, and you give it a name, there will be braces that enclose what that function does. You can even have more braces within the parent braces that can also hold more information.

Single Quotes

Single quotes and double quotes are bothed used for containing strings. You can basically use either, however, if you want to use quotes within a string you will use single quotes to contain the string.

Double Quotes

Just like single quotes you can use double quotes for a string (I prefer to use double quotes for my strings) however, there are times where you will definately want to use double quotes and that's when you will use a single quote within the string for words like "don't".
