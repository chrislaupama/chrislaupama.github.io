What is grid based design?

Grid based design is where you segment the design of the website into a grid, much like box design, in fact they are relatively similar.

Whats all the hype about responsive webpages?

Responsive webpages is all about making your website look and perform in a way that is aesthetically pleasing regardless of what device you are viewing the page from. So if you're looking at the page on a mobile device, desktop or even a TV it will still look good.

What is semantic structure?

The semantic structure is thinking about the semantic tags you will use to start converting your design into code. It's the next step you would take after slicing the design up into grids. This way you are able to practically read the visual design more like a blueprint.
